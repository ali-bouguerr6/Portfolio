Automated Job Offers Collection & Email Reporting

Academic Project | University of Strasbourg

R Web Scraping Gmail API

Built an end-to-end automation pipeline to collect job offers (Glassdoor), clean and format the data, and send a daily HTML email report. The workflow is scheduled for hands-free monitoring.

GitHub – Automated Job Offers Collection

AI Maintenance Agent

Academic Project | University of Strasbourg

Python LLM RAG Chroma DB

Designed an AI agent for maintenance support: analyzing machine data for anomaly/risk reasoning, and generating natural language explanations using LLM + RAG to retrieve relevant maintenance procedures.

GitHub – AI Maintenance Agent

Smart Gmail Sorting with Local LLM

Academic Project | University of Strasbourg

Python Ollama Gmail API

Implemented an email classification system using a local LLM (Ollama + Mistral) to automatically categorize Gmail messages (jobs, shopping, ads, others) and keep the inbox organized.

GitHub – Smart Gmail Sorting

Air Passenger Traffic Forecasting (Time Series)

Academic Project | University of Strasbourg

Forecasted monthly air passenger volumes using time-series analysis to capture trend and seasonality, and visualized predictions vs historical values to evaluate model performance.
